Getting Started

1

Open Developer Tools

Right-click anywhere on a webpage and select Inspect, or use the keyboard shortcut.

F12 or Ctrl + Shift + I or Cmd + Option + I
2

Locate the "WebAI" Panel

Look for the WebAI tab in the top navigation bar of DevTools. If you don't see it, click the double arrows (>>) to show more tabs.

3

Verify Environment

Check the Overview tab to ensure Built-In AI APIs are available and the required flags are enabled.
